## Installation:

1. Clone the repo
```sh
git clone https://github.com/jdgc14/pl-dle-client.git
```
2. Go to directory project
```sh
cd pl-dle-client/
```
3. Install NPM packages
```sh
npm install
```
4. Run project
```sh
npm run dev
```

## Dependencies:
**React JS ⚛️**, and use tools like **React Router**, **Bootstrap**.
